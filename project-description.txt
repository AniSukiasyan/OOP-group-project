Our project QuizGame is an interactive game in Java. The goal is to create a fun and challenging way to test knowledge using Object-Oriented Programming. It will work by pulling a random set of questions from a large bank of 100 to 200 questions stored in a text file. This means every time you play, the experience will be different. We will use classes to manage everything from the player's score to the different types of questions (like Multiple Choice or True/False).
